# IURA Plugin (NL)

AI-plugin voor Nederlandse juridische teams, ontworpen voor [Cowork](https://claude.com/product/cowork) — werkt ook in Claude Code.

Geïntegreerd met **IURA MCP** — juridische kennisbank voor Nederlands recht.

> **Disclaimer:** Deze plugin ondersteunt juridische workflows maar vormt geen juridisch advies. Controleer altijd conclusies met gekwalificeerde juristen.

## Status

Deze plugin is **in voorbereiding**. De IURA MCP connector voor Nederlands recht is geconfigureerd en wordt automatisch beschikbaar wanneer de service live gaat.

## Installatie

```
claude plugin install iura-nl@iura-plugins
```
